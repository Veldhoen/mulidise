Frankfurt ( Reuter ) - Der Umsatz mit Aktienoptionen an der DTB hat am Mittwoch erneut deutlich zugelegt . 
davon entfielen 47.417 ( 30.292 ) auf Calls und 22.821 ( 18.870 ) auf Puts . 
die Call-Put-Ratio betrug 2,07 ( 1,60 ) . 
Umsatzspitzenreiter waren erneut Siemens mit 12.156 ( 8555 ) Calls und 6647 ( 6113 ) Puts . 
auf den Dax wurden bis 16.00 Uhr 58.488 ( 35.554 ) Calls und 50.505 ( 64.987 ) Puts gehandelt . 
der März-Dax-Future notierte bis dahin nach einem Umsatz von 18.283 ( 16.832 ) Kontrakten mit 2916,5 ( 2890 ) Punkten . 
der März-MDax-Future lag nach einem Umsatz von 346 ( 328 ) Kontrakten bei 3028 ( 2990 ) Punkten . 
der Dax-Volatilitätsindex fiel auf 15,45 ( 16,29 ) . 
im März-Bund-Future wurden bis 16.00 Uhr 63.734 ( 76.694 ) Kontrakte umgesetzt , im März-Bobl-Future 74.468 ( 56.873 ) Stück . 
im Einmonats-Libor-Future gingen über alle Sichten 587 ( 399 ) Kontrakte um . 
auf den Bund-Future wurden 1102 ( 289 ) Call-und 643 ( 1360 ) Put-Optionen gehandelt , auf den Bobl-Future 2418 ( 2955 ) Calls und 2312 ( 1474 ) Puts .
